# Vite + React + Storybook
Clean and basic setup for **Vite** with **React** and **Storybook**.   

### Includes:
- Alias folders for both Vite & Storybook
- SASS/SCCS support

### Install:  
`yarn`

### Use:
`yarn dev`  
`yarn storybook`

### Alias:
Use @ to target `./src` folder:
```
import { Button } from '@/stories/Button';
<Button>This is a button</Button>
```
### Alias config:
`./storybook/main.js`  
`vite.config.js`
