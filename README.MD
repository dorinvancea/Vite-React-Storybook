# Vite + React + Storybook

Basic setup for:
Vite React: `https://vitejs.dev/`
Storybook: `https://storybook.js.org/`

Install:
`yarn`

Use:
`yarn dev`
`yarn storybook`

Alias:
Use @ to target `./src` folder:
```
import { Button } from '@/stories/Button';
<Button>This is a button</Button>
```
Alias config:
`./storybook/main.js`
`vite.config.js`
